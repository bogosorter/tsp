TSP Challenge

In this challenge, you are given a random set of points. Your goal is to determine the best possible TSP tour order and submit it to the evaluator. Your submission will be scored and added to the ranking.

The first line of the input has a single value n, the number of points, and each of the remaining lines has two integers, the point's x and y coordinates. Your submission should consist of n space-separated values. Each value represents the index of the point in the corresponding position of the TSP tour. Example Input; Example Solution.
